## 3. 下一阶段统一控制协议（单参数占比）

为避免多权重竞争解释歧义，下一阶段默认采用：

- `evaders.align_control_mode = "share"`
- `angle_noise = 0.0`
- 扫描单参数 `w_align = λ ∈ [0,1]`，其中：
  - `λ=1`：完全按邻域对齐项；
  - `λ=0`：完全按非对齐复合项（avoid/goal/explore/sep）；
  - 中间值线性混合。

非对齐复合项内部权重固定为：

- `w_goal = 1.2`
- `w_avoid = 1.5`
- `w_explore = 0.4`
- `sep_strength = 1.0`

默认配置文件：`configs/walign_share_noise0.toml`。

## 4. 下一轮最小实验矩阵（先清晰后扩展）

### 阶段 S1（结构性判断）

- `speed_ratio ∈ {0.9, 1.1, 1.3, 1.4}`
- `w_align ∈ {0, 0.05, ..., 1.0}`
- seeds `0..79`
- steps `600`

输出：

- `safe_frac`、`chi`、`chi_local`、`tau_P_ar1`、`xi_fluct`
- 原始散点 + 分层散点 + 分箱中位数带

### 阶段 S2（不稳定区扩样）

对 S1 中以下情形扩到 seeds `0..199`：

1. 最优 `w_align` CI 重叠严重；
2. `corr(safe,chi)` 符号不稳定；
3. 高压层（`speed_ratio>=1.3`）出现平台态。

## 5. 报告规范（本路线专用）

每次路线报告必须同时给出：

1. 原始点云结论（单次运行层）；
2. 格点均值结论（多 seed 聚合层）；
3. 二者是否一致；若不一致，说明随机性与分层因素。

避免仅使用单一层级图像下结论。
