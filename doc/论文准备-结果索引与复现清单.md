# 论文准备：结果索引与复现清单（2026-02-06）

## 1. 建议作为主文结果的实验集合

### 1.1 `w_align` 主线（任务内临界判定）

- 数据目录：`runs/sweep_20260206_merged_200seeds_grid`
- 报告目录：`doc/results_20260206_walign_task_internal_200seeds`
- 聚合表：`doc/results_20260206_walign_task_internal_200seeds/group_summary.csv`
- 主结论：
  - 在 `v_p/v_e=1.0~1.2` 范围内，`safe` 与 `chi`、`tau_P_ar1`、`xi_fluct` 多为正相关。
  - 峰值更像“区间最优”而非稳健单点最优。

### 1.2 追捕压力对比（关系边界）

- 数据目录：`runs/sweep_20260206_merged_pressure_sr14_240_grid`
- 报告目录：`doc/results_20260206_walign_pressure_091314_sr14_240seeds`
- 聚合表：`doc/results_20260206_walign_pressure_091314_sr14_240seeds/group_summary.csv`
- 主结论：
  - 中等压力下（`v_p/v_e≈1.0~1.3`）近临界优势可观察。
  - 高压力下（`v_p/v_e=1.4`）该关系明显减弱/失效。

### 1.3 噪声路线交叉对照

- 相变识别（无追捕/无安全区）：`doc/results_20260206_phase_noise_100seeds_steps1200`
- 任务噪声扫描（`w_align=1.0`, `speed_ratio=1.1`）：`doc/results_20260206_task_noise_w10_sr11_100seeds`
- 用途：
  - 支撑“任务内相关性”与“外部相变点迁移”是两个不同命题。

## 3. 复现命令模板

### 3.1 环境与检查

- `uv sync`
- `uv run pytest -q`
- `uv run pyright`
- `uv run chasecrit bench-speed --config configs/base.toml --iterations 256 --steps 240 --repeats 3 --seed 0 --json-out doc/bench_20260206_numba_speed_v2.json`

### 3.2 主实验命令（示例）

- `w_align` 主线（200 seeds 分两段执行后合并）：
  - `uv run chasecrit sweep-grid --config configs/base.toml --speed-ratio 1.0 1.05 1.1 1.15 1.2 --w-align 0 0.05 0.1 0.15 0.2 0.25 0.3 0.35 0.4 0.45 0.5 0.55 0.6 0.65 0.7 0.75 0.8 0.85 0.9 0.95 1.0 --seeds 0 1 ... 199 --steps 600 --no-save-events --no-save-timeseries --jobs 8`
- 压力对比（含 `sr=1.4` 扩样）：
  - `uv run chasecrit sweep-grid --config configs/base.toml --speed-ratio 0.9 1.3 1.4 --w-align 0 0.05 0.1 0.15 0.2 0.25 0.3 0.35 0.4 0.45 0.5 0.55 0.6 0.65 0.7 0.75 0.8 0.85 0.9 0.95 1.0 --seeds 0 1 ... 239 --steps 600 --no-save-events --no-save-timeseries --jobs 8`
- 报告生成：
  - `uv run chasecrit report --sweep-dir runs/<sweep_dir> --out-dir doc/results_<name> --title "<title>"`
