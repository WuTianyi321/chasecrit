# 项目目的（概述）

本项目研究的问题是：**逃跑者集群处于近临界态/临界态时，是否能在追逃任务中带来性能优势**。研究对象主要聚焦于**逃跑者集群**的局部交互与集体决策行为，追捕者策略可从简单到复杂逐步增强。

## 研究路线

1. **参数扫描路线（Near-critical tuning）**
   - 选取一种集群逃跑/集群运动算法，其中包含一个可调参数（例如从众/对齐强度 `w_align`，或噪声强度 `η`）。
   - 扫描控制参数并计算临界性相关统计指标（序参量、易感性、相关长度等），定位近临界区域。
   - 在追逃任务中评估性能指标（存活数、成功率、到达时间、拥挤浪费等），检验“性能峰”与“临界区”是否重合或存在系统偏移。

2. **自组织临界路线（SOC / self-tuning，后置主线）**
   - 设计个体基于局部观测的自调参策略，使群体在外部驱动与耗散下形成自组织（准）临界状态。
   - 在场景变化下检验鲁棒性，并与固定参数最优解对照。

## 场景与任务需求（导向）

为体现临界态潜在优势，任务场景应具备：

- **信息局部性与不确定性**：个体只能局部感知邻居动作/位置、追捕者信息与环境线索。
- **探索-利用权衡**：例如多个安全区、容量限制、局部可见与动态变化，使过度有序或过度无序都不占优。
- **可量化目标**：限定时间窗，明确成功/失败定义并可复现（随机种子、参数化配置）。
